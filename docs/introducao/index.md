# Bem-vindo(a) ao HIVE

Bem-vindo(a) à documentação oficial do **HIVE** — projeto de TCC desenvolvido para a ETS (Escola Técnica do Setor) da Bosch.

## O que é o HIVE?

O HIVE é uma **plataforma colaborativa de componentes de interface (UI)** que, em conjunto com um plugin desenvolvido para o Figma, visa reunir componentes criados na ferramenta de design juntamente com suas respectivas versões em diferentes linguagens e frameworks.

A solução possibilita a **conexão direta entre design e desenvolvimento**, promovendo maior organização e acesso aos recursos padrão do setor utilizados na construção de interfaces de diferentes projetos.
